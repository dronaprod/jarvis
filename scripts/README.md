# Jarvis Scripts

This directory contains build and installation scripts for Jarvis.

## Scripts

- **`build.sh`** - Build binary for current architecture
- **`build-universal.sh`** - Build universal binary (arm64 + x86_64)
- **`prepare-release.sh`** - Prepare release archives
- **`install_jarvis_user.sh`** - Install jarvis for current user

## Usage

### Build Binary
```bash
./jarvis/scripts/build.sh
```

### Build Universal Binary
```bash
./jarvis/scripts/build-universal.sh
```

### Prepare Release
```bash
./jarvis/scripts/prepare-release.sh
```

### Install for User
```bash
bash jarvis/scripts/install_jarvis_user.sh
```
